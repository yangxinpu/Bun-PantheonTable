# 🧙‍♂️ Avalon Online Server（阿瓦隆桌游服务端）

一个基于 **Bun + Elysia + MySQL** 的阿瓦隆（Avalon）桌游在线对战服务端项目，目标是实现 **房间制多人桌游、实时交互、完整对局记录与统计**。

> 本项目目前处于 **持续开发中**，主要用于学习与实践 **现代 Web 后端架构、实时通信与游戏状态管理**。

---

## 📖 项目背景

**阿瓦隆（The Resistance: Avalon）** 是一款经典的多人阵营推理桌游，具有以下特点：

- 多角色（梅林、刺客、忠臣、爪牙等）
- 隐藏身份
- 多阶段流程（组队 → 投票 → 任务 → 刺杀）
- 强状态一致性要求

本项目旨在将其实现为 **可扩展、可维护、工程化** 的在线版本服务端。

游戏规则参考文章：  
👉 https://zhuanlan.zhihu.com/p/685062276

---

## 🎯 项目目标

- ✅ 提供稳定的 HTTP / WebSocket 服务
- ✅ 支持用户系统（注册 / 登录 / 状态管理）
- ✅ 支持房间创建与玩家加入
- ✅ 支持完整阿瓦隆游戏流程
- ✅ 记录对局数据，统计胜率
- ✅ 提供清晰日志，方便调试与维护

### 核心技术

| 技术           | 说明                     |

## 📂 项目结构（规划）

```text
src/
├── index.ts              # 应用入口
├── routes/               # 路由模块
│   ├── users/             # 用户相关接口
│   ├── rooms/             # 房间相关接口
│   └── game/              # 游戏流程接口
├── services/              # 业务逻辑
├── plugins/               # 插件（日志等）
├── db/                    # 数据库连接与模型
├── types/                 # 公共类型定义
└── utils/                 # 工具函数
````
